# ESI SmartFarming — TP POO 2025/2026

## Prérequis

- **JDK 17+** : https://www.oracle.com/java/technologies/downloads/
- **Maven 3.6+** : https://maven.apache.org/download.cgi

> Maven télécharge JavaFX automatiquement — aucune installation supplémentaire requise.

## Lancer le projet

```bash
mvn javafx:run
```

C'est tout. La première exécution télécharge les dépendances (~30 secondes), les suivantes démarrent immédiatement.

## Vérifier les prérequis

```bash
java -version    # doit afficher 17 ou supérieur
mvn -version     # doit afficher 3.6 ou supérieur
```
